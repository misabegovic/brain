# Codeplain — $3M seed, June 2026

Slovenian startup (Ljubljana / Berlin / San Francisco). Raised a $3M
seed led by GapMinder with Silicon Gardens (announced late June
2026). Founders: Dusan Omercevic (previously founded Cleanshelf,
enterprise SaaS management, acquired by SAP LeanIX in 2021) and
Predrag Radenkovic. Notable advisor: Johan Rosenkilde, creator of
the SpecLang research project at GitHub Next.

**Product/thesis:** an AI platform that generates production-ready
code from plain-language software specifications. The primary
software artifact becomes the specification; "code should not be
maintained — code should be regenerated. Specs should be reviewed,
and it's the specs that you maintain." AI agents create and evolve
specifications, which are automatically transformed into tested,
validated, production-ready software. Cited technical basis:
"Phoenix Architecture" (Chad Fowler) — source code is no longer the
long-lived asset, continuously regenerable from well-defined specs.

**Funding use:** develop the platform, helping AI agents create
higher-quality specifications and transform them into reliable,
secure, production-ready software.

**Stage:** seed; no public traction figures, spec-format details, or
tooling specifics (CLI/platform/integrations) in the covered
articles. The New Stack piece (2026-06-25, Paul Sawers) is
paywalled/blocked from automated capture; headline captured.
